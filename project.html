Game Lazy Bee

The game is a side-scroller where the player controls a bee, and flies between spiders and nests without hitting them. Designed and integrated custom pixel art sprites, audio effects, and UI elements. Implemented the game mechanics with Collider 2D, boundary detection, spawn prefabs and gravity. Applied animations and parallax effects to the game components to enhance user experience. Keyboard or mouse-driven player, by avoiding movable barriers and gaining scores.
